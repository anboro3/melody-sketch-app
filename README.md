# Melody Sketch 🎹

ブラウザだけで動作する、軽量なピアノロール・シーケンサーです。
アイデアを即座に形にするための「スケッチブック」として設計されています。

![Screenshot](screenshot.png)
*(スクリーンショットはここに配置)*

## 特徴 ✨

- **完全ブラウザ動作**: インストール不要。`index.html` を開くだけで動作します。
- **ピアノロール編集**: 直感的なグリッドインターフェース。
  - **Drawモード**: 鉛筆ツールで音符を配置。
  - **Eraseモード**: 消しゴムツールで音符を削除。
- **リッチな音源**:
  - **Piano**: FM合成によるエレクトリックピアノ風サウンド。
  - **Basic Waves**: Sine, Triangle, Square, Sawtooth波形も選択可能。
- **スケールサポート**: KeyとScale（Major, Minor, Dorianなど）を指定すると、スケール外の音がグレーアウトされ、作曲を補助します。
- **自動保存**: オートセーブ機能により、リロードしても作業内容が失われません。
- **MIDIエクスポート**: 作成したメロディを `.mid` ファイルとしてダウンロードし、DAWで使用可能。

## ショートカット ⌨️

- **Space**: 再生 / 一時停止
- **Z**: 先頭に戻る (Rewind)
- **Ctrl + Z**: 元に戻す (Undo)
- **D**: Draw (鉛筆) ツールに切り替え
- **E**: Erase (消しゴム) ツールに切り替え

## 使い方 🚀

1. リポジトリをダウンロード（またはClone）します。
2. フォルダ内の `index.html` をモダンなWEBブラウザ（Chrome, Edge, Firefoxなど）で開きます。
3. 画面をクリックしてAudioContextを有効化し、作曲を開始してください！
